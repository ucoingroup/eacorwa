# EACO RWA · Global Ports & Hotel Asset Tokenization

> EACO Earth Port RWA — Global Asset Digital Circulation Ecosystem

## 🌍 Overview

EACO RWA is a concept verification (PoC) platform demonstrating how real-world assets (commercial buildings, budget hotel apartments) can be tokenized through Hong Kong licensed institutions and distributed globally via 10 neighboring land ports.

## 📋 Features

- **Multi-language**: Chinese ↔ English toggle
- **10 Global Ports**: Vietnam, Myanmar, Laos, Kazakhstan, Russia, Mongolia, North Korea, Nepal, Pakistan, Kyrgyzstan — with risk level labels
- **10 Hotel RWA Methods**: STO tokenization, fragmented stay NFTs, DeFi staking pools, cross-border trade vouchers, supply chain tokenization, DAO governance, loyalty on-chain, ESG carbon tokens, cross-border REITs, data assetization
- **Token Information Cards**: CA, chain, economics, RWA scenarios
- **Action Roadmap**: 4-phase implementation plan
- **Responsive Design**: Mobile-friendly
- **Compliance Disclaimer**: PoC only, HK licensed institution pathway

## 🛠️ Tech Stack

- Pure HTML/CSS/JS (no dependencies)
- Single-file SPA architecture
- CSS custom properties for dark theme
- i18n system with data-attributes
